# herms-frontend


### Quick start
**Make sure you have Node version >= 5.0 and NPM >= 3**

```bash
# clone our repo
git clone https://github.com/tibysko/herms-frontend.git

# change directory to our repo
cd herms-frontend

# install the repo with npm
npm install

# start dev
npm start

# build for production
npm run build

# run tests
npm test
```
